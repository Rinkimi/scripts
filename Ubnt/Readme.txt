Для работы скрипта необходимо установить python 3 (https://www.python.org/ftp/python/3.6.1/python-3.6.1.exe)
При установке python обязательно нужно поставить галочку "Add Python 3.6 to PATH"
Разархивировать скрипт, network.bat и list.csv файл в директорию C:\Ubnt\
Запустить network.bat файл от имени администратора.
Ввести в появившемся окне номер рамки, нажать Enter.
Подключить одну Wi-Fi станцию к ноутбуку, а вторую во внутреннюю сеть рамки.
Как только на Wi-Fi станции станут активны диоды индекаторов уровня Wi-Fi сигнала(4 правых индикатора на станции) можно работать с внутренней сетью рамки.
